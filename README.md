# 🃏 Go Phish: Strict Edition

A professional-grade, web-based "Go Fish" card game built with **Python (Flask)** and **Modern JavaScript**. This version uses "Strict Rules" and features a **Smart AI** with a memory engine.

## ✨ Features
- **Strict Rule Engine**: Automatic hand refills and "Lucky Catch" extra turns.
- **Smart AI**: The computer remembers which cards you've asked for and will target you strategically.
- **Responsive UI**: Playable on Desktop, Tablets, and Mobile phones.
- **Local Multiplayer**: Host on your PC and play with family on the same Wi-Fi.

## 🛠️ Tech Stack
- **Backend**: Python 3.x, Flask
- **Frontend**: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6)
- **Deployment**: Localhost / PythonAnywhere
